# Yarem-Streaming-Projects

## Setup

To set up in OBS, please download the folling plugin

``` https://obsproject.com/forum/resources/xobsbrowserautorefresh-timed-automatic-browser-source-refreshing.1677/ ```

Inside of filters for the scene, make sure the scene is the browser, and then add a Chroma Filter for green and a Auto Browser Refresh every 1 second

---

In Addition, please look at the streamer.bot settings tab in actions and make sure to click the test trigger after each one is changed.
